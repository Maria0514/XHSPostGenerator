# 小红书时尚穿搭文案创作指南

## 一、穿搭内容类型

### 1. 穿搭灵感分享
展示不同风格、场景的穿搭搭配，为粉丝提供穿衣灵感。

**核心要素**：
- 整体look展示
- 单品信息（品牌、价格）
- 搭配思路说明
- 适用场景
- 身材体型建议

### 2. 衣橱管理
教授如何整理衣橱、建立胶囊衣橱、实现极简穿搭。

**核心要素**：
- 断舍离方法
- 基础单品清单
- 配色方案
- 多套搭配演示
- 季节过渡技巧

### 3. 身材优化穿搭
针对不同身材特点，提供显瘦、显高、扬长避短的穿搭方案。

**核心要素**：
- 身材分析
- 单品选择建议
- 搭配技巧
- 前后对比
- 避坑指南

### 4. 单品测评与推荐
深入测评某一品类单品，提供购买建议。

**核心要素**：
- 产品信息
- 面料质感
- 版型尺码
- 上身效果
- 性价比分析

```

✅ 高腰款（显腿长）

❌ 紧身裤/哈伦裤（暴露缺点）

✅ 中长款（遮大腿）
✅ 高腰款（提升腰线）

💡 竖条纹：视觉显瘦显高

❌ 不要穿低腰裤（显腿短）

```

```
件数+效果+适用对象

【什么是胶囊衣橱】
概念：用少量百搭单品，搭配出多种look
优点：
✅ 减少选择困难
✅ 提升搭配效率
✅ 节省预算
✅ 衣柜整洁

【基础单品清单】（秋季版）
📦 上衣（8件）
1. 白色t恤×2
2. 黑色t恤×1
3. 针织衫（奶茶色）×1

5. 衬衫（蓝色）×1
6. 西装外套（黑色）×1

📦 下装（6件）
1. 牛仔裤（蓝色）×1
2. 黑色直筒裤×1
3. 卡其色阔腿裤×1
4. A字半裙（黑色）×1

📦 外套（3件）
1. 风衣（卡其色）×1

3. 羽绒服（黑色）×1

📦 鞋包（3双+2个）
1. 小白鞋×1
2. 乐福鞋×1
3. 短靴×1
4. 托特包（黑色）×1
5. 腋下包（棕色）×1

【配色原则】
基础色：黑白灰（占70%）
大地色：卡其、奶茶、棕色（占20%）
点缀色：蓝色或其他（占10%）

【搭配示范】
Week 1:
周一：白tee+黑色西装裤+乐福鞋
周二：蓝衬衫+牛仔裤+小白鞋

周末：白tee+阔腿裤+风衣

（可以继续搭配出更多组合）

【季节过渡技巧】
初秋：
- 单穿：t恤、衬衫
- 外搭：薄外套

深秋：
- 叠穿：衬衫+针织衫
- 外搭：风衣、夹克

入冬：
- 内搭：打底+毛衣
- 外搭：羽绒服

【购买建议】
💰 预算分配：
- 基础款：占70%预算（买质量好的）
- 时尚款：占30%预算（跟随潮流）
```

## 四、拍照技巧与图片构图

### 1. 全身照拍摄
```
【角度】
- 低角度拍摄（显腿长）
- 手机放在胸部高度（黄金比例）
- 斜45度（显瘦显立体）

【姿势】
- 一只脚在前（显腿长）
- 手自然垂放或插口袋（不僵硬）
- 身体微侧（显瘦）
- 不要直视镜头（自然）

【背景】
- 纯色墙壁（突出服装）
- 街景（增加氛围感）
- 绿植（清新自然）

【光线】
- 自然光最佳
- 阴天柔和
- 避免正午阳光（太硬）
```

### 2. 半身照细节
```
【构图】
- 从腰部开始拍（显腿长）
- 留白天空（高级感）
```
